There are 33 letters in Russian alphabet.
We will use yellow color to indicate vowel letters. Likewise blue color will be used to indicate consonant letters.


Let's read paired vowels and consonants.


Some voiced consonants have paired dull consonants. Let's compare them.

Letters "Я" "Ё" "Ю" "И" "Е" add softness to the preceding consonant letter.
Letters "А" "О" "У" "Ы" "Э" make the preceding consonant hard.

МЯ - МА, ЛЁ - ЛО, РЮ - РУ, ЗИ - ЗЫ, ПЕ - ПЭ

Letters "Ж" - "Ш" make hard sound. Use soft vowel "И" instead of hard "Ы" when you write "ЖИ - ШИ"

МА-ШИ-НА, ШИШ-КА, У-ШИ, Ё-ЖИК, У-ЖИН, ЖИ-РАФ

Letters "Ч" - "Щ" make soft sound. Do not write them with soft vowels "Ю" and "Я". Instead, use letters "У" and "А". ЧА-ЩА and ЧУ-ЩУ!

ДА-ЧА, ТУ-ЧА, ЧАЙ-НИК, ЧА-ЩА, ЩА-ВЕЛЬ, ЧУ-ДО, ЧУ-ЛАН, ЩУ-КА
